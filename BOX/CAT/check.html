HEX (Formerly CHECK-AI)

HEX is a planned activity in which students; over the course
of a term make a javascript AI that plays the board game HEX. The AI, at the end of the term are then pitted against each other, competing for the secret prize.

PREREQUISITES

It is recommended you have a basic understanding of:

  • Programming Concepts
  • Maths
  • Working in small teams
  • Javascript knowledge a plus

FRAMEWORK

We have developed HEX as a web app, so you can go over to that and start practicing.
